# Assets

This directory contains visual assets for the marketplace listing.

## Required Files

| File | Size | Description |
|------|------|-------------|
| `icon.png` | 256x256 | Plugin icon (square) |
| `banner.png` | 1280x640 | Marketplace banner |
| `screenshots/dashboard.png` | 1280x720 | Dashboard screenshot |
| `screenshots/ralph-loop.png` | 1280x720 | Ralph loop in action |
| `screenshots/init-wizard.png` | 1280x720 | Init wizard screenshot |

## Design Guidelines

### Icon
- Square format (256x256 px)
- Simple, recognizable design
- Works on light and dark backgrounds
- Suggested: TDD cycle icon (red-green-refactor) with Ralph character

### Banner
- 1280x640 px
- Include plugin name and tagline
- Show key features visually
- Use brand colors

### Screenshots
- 1280x720 px (16:9 ratio)
- Actual product screenshots
- Annotate key features if helpful
- Dark or light theme (match common usage)
